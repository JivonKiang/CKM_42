# CKM_42

Cardiovascular-Kidney-Metabolic Syndrome Analysis Project

This repository contains the analysis code for CKM (Cardiovascular-Kidney-Metabolic Syndrome) research.

## Project Structure

- **Raw Data Processing/**: Contains scripts for data processing from CHARLS and NHANES datasets
  - CHARLS: Chinese Health and Retirement Longitudinal Study data processing
  - NHANES: National Health and Nutrition Examination Survey data processing

- **Statistics and Visualization/**: Contains statistical analysis and visualization scripts
  - Cox regression analysis
  - RCS (Restricted Cubic Splines) analysis
  - KM (Kaplan-Meier) survival analysis
  - Various visualization scripts

## Requirements

R statistical software with required packages.

## Note

Large data files (.RDS, .RData, .xlsx, .csv, etc.) are excluded from this repository via .gitignore due to size constraints.
